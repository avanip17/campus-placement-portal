# Campus Placement Portal

A centralized web-based application to streamline university recruitment operations, managing interactions between Admins, Recruiters, and Students.

# Features

* Role-Based Access: Distinct dashboards for Admins, Recruiters, and Students.
* Student Portal: Register, browse job opportunities, submit applications, track interview statuses in real-time, and manage profiles.  
* Recruiter Portal: Post new job openings, review submitted student applications, and manage the hiring pipeline.  
* Admin Dashboard: Manage company recruiters, schedule placement drives, export student data, and monitor system statistics.  
* Application Tracking System: Complete cycle from job posting to application submission and final placement status. 

# Tech Stack

* Backend: Python, Flask
* Database: SQLite (SQLAlchemy ORM)
* Frontend: HTML, CSS (Bootstrap 5), Jinja2 Templating
* Authentication: Flask-Login
